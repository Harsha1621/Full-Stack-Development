What is Web Storage?

localStorage and sessionStorage are part of the Web Storage API in HTML5. They allow web applications to store data directly in a user's browser.

  • localStorage: Stores data with no expiration time. The data remains even after the browser is closed and reopened.
  • sessionStorage: Stores data for one session only. The data is lost when the browser tab is closed.

These are commonly used to save user preferences, session info, or temporarily store form data without needing a server.
